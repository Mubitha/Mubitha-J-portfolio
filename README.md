# 🌐 Modern Minimalist Portfolio Website

A highly responsive, premium dark-themed portfolio website designed for showcasing web development and programming expertise. Fully optimized for Mobile, Tablet, and Desktop screens.

---

## ✨ Features

- **Premium Dark Aesthetic:** Modeled after cutting-edge developer platforms like GitHub and Vercel.
- **100% Mobile Responsive:** Automatically adapts layout grids into elegant single-column structures on smaller screens.
- **Smart Contact Integration:** Replaced erratic default OS mail triggers with seamless browser-based Gmail compose redirects to maximize recruiter engagement.
- **Custom Interactions:** Includes dynamic dual-ring cursor tracking and smooth intersection-observer scroll fade-ins.
- **Dynamic Profile Metrics:** Showcases professional achievements, academic stats, and open-to-work target locations clearly using intuitive badge interfaces.

---

## 🛠️ Tech Stack Used

- **Front-End:** HTML5, CSS3 (Custom Variables & Flexbox/Grid Layouts)
- **Interactions & Logic:** Native Vanilla JavaScript (ES6+)
- **Typography:** Outfit & Cormorant Garamond via Google Fonts
